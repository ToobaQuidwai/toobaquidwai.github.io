# Dr. Tooba Quidwai Portfolio

Static portfolio site prepared for GitHub Pages.

## Files

- `index.html` - public homepage
- `assets/paper-previews/` - figure-page renders from local paper PDFs used in the publications section
- `final_version.html` - earlier draft kept for reference
- `version_1_draft.html` - earlier draft kept for reference

## Local preview

Open `index.html` directly in a browser, or run a simple static server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

If this folder is connected to a GitHub repository:

1. Push `index.html` and this `README.md` to the default branch.
2. In GitHub repository settings, enable Pages from the repository root on the default branch.
3. Wait for the Pages deployment to finish and verify the published URL.

## Notes

- Public contact is limited to one email address plus academic profile links.
- The pasted GitHub personal access token should be revoked and replaced before any deployment workflow uses GitHub credentials.
